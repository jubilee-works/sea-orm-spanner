# sea-orm-migration-spanner

Migration utility for [SeaORM](https://www.sea-ql.org/SeaORM/) with Google Cloud Spanner.

Provides a CLI and library for managing Spanner schema migrations using SeaORM's migration framework.

## Part of sea-orm-spanner

This crate is a component of [sea-orm-spanner](https://github.com/jubilee-works/sea-orm-spanner). See the main repository for full documentation.
